class LinkedArray<T>

Linked array, to do Camel like operations on array

To have an instance use emptyLinkedArray or linkedArrayOf

The U (for union) operation is the concatenation operation, by example :

a U [b, c, d]    ->  [a, b, c, d]
[b, c, d] U a    ->  [b, c, d, a]
[a, b] U [c, d]  ->  [a, b, c ,d]
